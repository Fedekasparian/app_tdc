App de gestión para instructora de fitness (mamá de Federico). Single user, web app responsive mobile-first.

**Why:** Simplificar la gestión diaria de clases, alumnas y pagos de la instructora.
**How to apply:** Todas las decisiones de diseño deben priorizar simplicidad de uso para un usuario no-técnico en celular.

Stack: Next.js + Supabase (Auth, DB PostgreSQL, Storage para videos)
Deadline: Antes de mayo 2026 (~3 semanas desde 2026-04-08)

MVP Semana 1-2: Alumnas (ABM + import Excel) + Clases fijas + Ejercicios con video
MVP Semana 3: Pagos + Reporte mensual básico
Post-MVP: Recomendaciones automáticas, análisis estadístico

Entidades: exercises, classes, sessions, students, student_classes, class_exercises, session_exercises, payments, settings
